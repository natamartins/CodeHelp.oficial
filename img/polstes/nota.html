Sintaxe A forma como a linguagem é escrita. Sempre verificar a sintaxe quando escrever o código.  Adicionando arquivo JavaScript “<script scr=”adicione o arquivo .js”>” é usado para ligar o arquivo .js no arquivo .html . Comentários “//” ignora o código que estiver escrito a frente. “/*” ignora  todas as linhas que estiverem entre */. “console.log(“”)” – escreve oq estiver entre parênteses no devtools. Tipos de Dados String String é uma cadeia de caracteres. Para escrever texto no JS temos que escrever entre ``- esse se usa quando tiver que escrever em multilinha ou quando for escrever valores - ou entre ‘’ ou entre “”. Number No JS temos 4 tipos de números, os inteiros (por exemplo: 13), os reais ou float (por exemplo: 83.1), o Not a Number (NaN) e o infinito (Infinity). Bolean São apenas dois valores – “true / flase” Undefined vs null Undefined = indefinido – algo que realmente não exite. Null = Nulo – algo que existe porém não tem nada dentro dele. Object  Object é um dado estrutural. Todo objeto possui propriedades/atributos e  funcionalidades/métodos. { propriedade: “valor” } – um objeto é escrito dessa forma.  Array Array é um dado estrutural. É uma lista, um agrupamento de dados. Se escreve entre []. Variáveis Conhecendo Variáveis Nomes simbólicos para receber algum valor. Atalhos de códigos. Identificadores. 3 palavras reservadas para criar uma variável: - var – pode mudar o valor durante a aplicação  -let - pode mudar o valor durante a aplicação  -const – não pode mudar o valor da variável durante a aplicação Scope e var Determina a visibilidade da variável. Block Statement – é uma declaração de bloco. O bloco também criara um novo scope.  Para criar um Scope usamos – iniciamos o bloco com uma { colocamos qualquer código aqui dentro } fechamos um bloco. “var” – é global(é tudo que está dentro do script) e local(tudo que está dentro do Scope). Hoisting = elevação. Scope let e const  “let” – é apenas local. “const” – é local mas não conseguisse fazer variação do falar. Nomeando Variáveis JS é case-sensitive (sensível ao caso) – significa que letras maiúsculas e minúsculas importam. JS aceita cadeia de caracteres Unicode – acentuações e aspas. Posso: Iniciar com esses caracteres especias: $ _.  Iniciar com Letras.  Colocar acentos. Letras maiúsculas e minúsculas fazem diferença. Não posso: Iniciar com números. Colocar espaços vazios no nome. Ideal: Criar nomes que fazem sentido.  Que explique o que é a variável é ou faz.  camelCase snake_case Escrever em Inglês Functions Para declarar uma função faça: function nomeFunção() { código a ser executado}. Para executar uma função use: nomeFunção() Para colocar funções dentro de variáveis use: const variável = function() {}. Arguments e parameters São similares a variáveis que funcionam para colocar dados nas funções.  Parametros são utilizados dentro de () na declaração da função. Argumentos são utilizados dentro de () na execução da função. Retornando valores dentro da função  Usamos a palavra “return nome” dentro da função e trocamos o nome do console log pelo “nome = valor”. Function hoisting Se uma função é criada depois da execução, ela sofre automaticamente hoisting. Arrow function Uma arrow function é escrita da seguinte maneira: const nomeFuncao = () => {} Callback function Uma callback function é uma função que está sendo passada para outra função como parâmetro.
